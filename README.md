## 🚀 Features

- **Hero Section** — Animated name, title, CTA buttons, floating orbs
- **About Section** — Bio, animated avatar, stats, skill tags
- **Skills Section** — 6 skill cards with hover effects
- **Projects Section** — 6 projects with live links (Featured: Snap-to-Site AI)
- **Contact Section** — Email, GitHub, LinkedIn cards
- **Custom Cursor** — Smooth lagging purple cursor (desktop)
- **Scroll Reveal** — Elements animate in on scroll
- **Active Nav** — Highlights current section in navbar
- **Responsive** — Mobile, tablet, and desktop ready
- **Dark Theme** — Deep dark with purple accent throughout
